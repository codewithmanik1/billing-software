# More Jwellers — Billing System

A premium gold shop invoice management system built for **More Jwellers**.

## Key Features
- **Official Branding**: Integrated MJ logo and luxury gold theme.
- **Invoice Management**: Create, track, and print formal invoices.
- **Customer Directory**: Manage client profiles and billing history.
- **Reporting**: Real-time sales and payment insights.
- **Responsive Design**: Elegant dark/light mode with premium aesthetics.

## Tech Stack
- React + TypeScript
- Tailwind CSS
- Lucide React (Icons)
- Vite (Build Tool)
